Универсальный конвертер

Универсальный конвертер - это удобное приложение, разработанное на Python с использованием PyQt6, для быстрого и точного конвертирования физических величин и валют.

Особенности:

Конвертер физических величин

Конвертер валют

Технологии:

Python 3.10+

PyQt6 - графический интерфейс

requests - получение курсов валют (курсы вылют берутся с ЦБ РФ)

xml.etree.ElementTree - парсинг XML от ЦБ РФ

Структура проекта:

converter/

├── main.py              # Основной файл приложения

├── window.py            # Сгенерированный UI-код

├── converter.ui         # Файл интерфейса Qt Designer

├── converter_logic.py   # Логика конвертации

├── coefficients_and_categorys.py # Коэффициенты и единицы измерения

├── exeptions.py        # Пользовательские исключения

├── run.py              # Скрипт для горячей перезагрузки (для разработки)

└── README.md           # Документация

Для установки склонируйте репозеторий и запустите main.py. Убедитесь, что у вас утановленны библиотеки PyQt6 requests watchdog.
